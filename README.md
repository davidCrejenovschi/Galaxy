### 📦 Supported Platforms
- **Windows:** `.exe`
- **Android:** `.apk`
- **macOS:** `.dmg`

**Galaxy** is a space runner arcade game developed in Python using the Kivy framework. It features dynamic 3D perspective movement and procedural path generation.
